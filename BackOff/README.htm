BackOff                  PHP-FUSION THEME

Installation

 

  1. Copy the BackOff folder (BackOff/themes/BackOff) up upon your server in the folder Themes.
  2. Log on to your website.
  3. Go to the Administration-panel.
  4. Go to the menu System Admin.
  5. Go to Main Settings.
  6. Scroll down and set the setting Site theme to BackOff.

 

Editing logo and forum-images

In order to change/edit the logo and forum-images, you will need to have an editor that is able to handle PSD files, such as Photoshop.
